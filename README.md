## 概要

pi（対話/非対話）+ tmux + pi-gl-mem によるパイプライン型マルチエージェントシステム。
manager が役割設定・経路設定を行い、worker 同士が連鎖して処理し、最終結果のみ manager に報告する。

### クイックスタート

```bash
cd /mnt/c/projects/pi-multi-agent
bash pi-team-run.sh
  → チーム名を入力
  → ワーカー数 (1-4) を入力
  → tmux セッションが起動
```

### 依存関係

| コマンド | 用途 | 必須 |
|---------|------|------|
| pi | AI エージェント | ✅ |
| tmux | ペイン管理 | ✅ |
| pi-gl-mem | 記憶永続化 | ✅ |
| wt.exe | 別窓表示 | WSLのみ任意 |
