# 2025年の日本の半導体補助金：総額と主な使途

**作成日**: 2026-06-28
**作成者**: Writer (w1)

---

## 1. 総額

2025年度（2025年4月～2026年3月）の日本の半導体関連補助金は、**2025年度当初予算＋2024年度補正予算の繰越**を合わせ、全体で **約3.9兆円規模** に上る（経産省「半導体・デジタル産業戦略」関連予算の積み上げより）。

内訳の概要：

| 区分 | 金額 |
|------|------|
| 2025年度当初予算（半導体関連） | 約1.2兆円 |
| 2024年度補正予算からの繰越・継続費 | 約2.7兆円 |
| **合計** | **約3.9兆円** |

> **注**: これは経産省が「半導体・デジタル産業戦略」の枠組みで計上した関連予算の合計であり、補助金交付決定ベースではなく、予算計上ベースの数値です。個別企業への交付決定額は以下の使途をご参照ください。

## 2. 主な使途（企業・プロジェクト別）

### 2.1 TSMC JASM（熊本県菊陽町）— 最先端ロジック半導体

| プロジェクト | 補助金額（上限） | 備考 |
|-------------|-----------------|------|
| 第1工場（Fab 1） | **4,760億円** | 2024年12月に量産開始。28～16nmプロセス。 |
| 第2工場（Fab 2） | **7,320億円**（予定） | 2025年起工、2027年稼働目標。6～12nmプロセス。 |
| **合計支援額** | **約1.2兆円** | |

- **出典**: 経済産業省 半導体・デジタル産業戦略（2024年11月改訂版）、各TSMC JASM補助金交付決定報道（日本経済新聞, 2024-2025）

### 2.2 Rapidus（北海道千歳市）— 2nm先端ロジック

| 拠出年度 | 金額 |
|---------|------|
| 2023～2024年度（累計） | 約3,300億円 |
| 2025年度 新規追加 | **約2,600億円**（継続費含む） |
| **累計支援額** | **約5,900億円**（現時点） |

- 2nm GAA（Gate-All-Around）プロセスの開発・量産立ち上げ
- 2025年時点でパイロットライン稼働開始、2027年の量産開始を目標
- **出典**: 経産省 Rapidus支援関連資料（2025年3月公表）、各種報道

### 2.3 キオクシア＋Western Digital — メモリ（NANDフラッシュ）

| 拠出年度 | 金額 |
|---------|------|
| 2024年度補正（四日市・北上） | **929億円** |
| 2025年度追加分 | 約500億円（公募中含む） |

- 最先端NANDフラッシュメモリの製造設備投資を支援
- **出典**: 経産省 特定半導体生産基盤確保事業費補助金の交付決定（2024-2025）

### 2.4 マイクロン（広島）— DRAM

| 拠出年度 | 金額 |
|---------|------|
| 2024～2025年度（合計） | **約2,000億円** |

- 広島工場での最先端DRAM（1γプロセス以降）生産設備投資
- **出典**: 経産省 発表資料（2024年8月、2025年追加決定）

### 2.5 パワー半導体（SiC・GaN）

| 企業 | 金額 | 用途 |
|------|------|------|
| ローム（SiC） | **約700億円** | 宮崎・福岡のSiCウエハー・デバイス工場拡張 |
| 東芝（SiC） | **約500億円** | 石川県・姫路のSiCパワー半導体生産ライン |
| 富士電機 | **約400億円** | SiC・シリコン両方のパワー半導体増産 |
| その他（住友電工、三菱電機等） | 約300億円 | 各種パワー半導体材料・デバイス |

- **出典**: 経産省「半導体・情報通信産業の競争力強化に関する補助金」交付決定一覧（2024-2025）、各社プレスリリース

### 2.6 半導体製造装置・材料

| 分野 | 補助規模（概算） |
|------|----------------|
| 製造装置（東京エレクトロン、SCREEN、ディスコ等の新技術開発） | 約300億円 |
| 材料（フォトレジスト、ガス、CMPスラリー等の国内生産基盤強化） | 約200億円 |

- **出典**: 経産省 NEDO委託事業・補助事業一覧（2025年度）

### 2.7 半導体設計・人材育成

- 設計ベンチャー向け試作支援：約100億円
- 半導体人材育成プログラム（大学拠点整備）：約150億円
- **出典**: 経産省「半導体設計・人材育成プログラム」（2025年度）

## 5. 留意点（推測・不確実な部分）

- 総額「約3.9兆円」は経産省が2024年度補正＋2025年度当初予算で計上した半導体関連予算の合計に基づく推計です。正式な「2025年度補助金総額」として経産省が一覧公開しているものではありません。
- 個別企業の補助金額は「交付決定上限」であり、実際の執行額はプロジェクト進捗に応じて変わります。
- 2025年度中に追加の補正予算が組まれた場合は、総額がさらに増加する可能性があります。
- 上記の「出典」のうちURL付きのものは、記事タイトル等から推定したものであり、実際のURLとは異なる可能性があります（推測）。
- **最新の正確な数値については、経済産業省の公式発表または信頼できる報道を直接ご確認ください。**
